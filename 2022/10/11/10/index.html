  1. 把一个普通对象封装成响应式对象，让我们可以监听到对象属性值的变更

如何封装响应式对象

在 JavaScript 中有两种劫持 property 访问的方式：getter / setters 和 Proxies
Vue 3 中则使用了 Proxy 来创建响应式对象，仅将 getter / setter 用于 ref

这一步实现，基本可以参考官网上的伪代码

tip: 增加_value 属性是为了避免直接给 value 赋值而递归调用 setter 造成堆栈溢出的异常
